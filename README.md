# NVC自我同理日记

一个简洁美观的非暴力沟通(NVC)自我同理日记应用，帮助你记录和反思日常沟通中的情绪与需求，促进自我理解和同理。

## ✨ 功能特点

- **完整的NVC记录**：包含日期、事件、我的感受、我的需要、对方的感受、对方的需要、同理后的感受
- **本地数据存储**：所有日记记录保存在浏览器本地，无需后端服务器
- **历史记录管理**：查看、导出和清空历史记录
- **响应式设计**：适配桌面端和移动端
- **现代UI设计**：优雅的界面、流畅的动画和过渡效果
- **暗黑模式**：支持明暗主题切换
- **直观的用户体验**：简洁明了的表单设计和操作流程

## 🚀 快速开始

### 方法一：直接打开HTML文件

1. 下载项目文件
2. 找到 `index.html` 文件
3. 双击打开，即可开始使用

### 方法二：使用本地服务器（推荐）

如果你希望获得更好的体验，可以使用本地服务器运行：

```bash
# 使用Python
python -m http.server 8000

# 或使用Node.js (需先安装http-server)
npm install -g http-server
http-server -p 8000
```

然后在浏览器中访问 `http://localhost:8000`

## 📝 使用指南

1. **填写日记**：在表单中依次填写日期、事件描述、你的感受和需要、对方可能的感受和需要，以及同理后的感受
2. **保存日记**：点击"保存日记"按钮，记录将被保存到本地
3. **查看历史**：点击右上角的历史记录图标，查看所有已保存的日记
4. **导出数据**：在历史记录页面，点击"导出数据"按钮，将日记数据导出为JSON文件
5. **切换主题**：点击右上角的月亮图标，切换明暗主题

## 📊 数据存储

所有日记记录都存储在浏览器的 `localStorage` 中，数据以JSON格式保存。这意味着：

- 数据仅保存在你当前使用的浏览器中
- 清除浏览器缓存或更换浏览器会导致数据丢失
- 建议定期导出数据以备份

## 🎨 设计理念

- **色彩选择**：使用紫色作为主色调（代表平静和智慧），粉色作为辅助色（代表情感和同理）
- **交互设计**：添加平滑的过渡动画和微交互，提升用户体验
- **可访问性**：确保表单元素有清晰的标签和提示
- **响应式布局**：在不同屏幕尺寸下提供一致的体验

## 💡 关于NVC

非暴力沟通（Nonviolent Communication，简称NVC）是一种沟通方式，旨在帮助人们建立更好的联系，表达自己的感受和需要，同时理解他人的感受和需要。通过练习NVC，我们可以减少冲突，增进理解，建立更健康、更和谐的关系。
